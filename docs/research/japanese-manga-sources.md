## MVP 實作策略

### 方案：WebView 瀏覽器 + OCR 整合

由於各平台有 DRM 保護，不適合直接抓取圖片。最務實的方式是：

1. **WebView 內嵌瀏覽器** — 在 app 內開啟日本漫畫網站
2. **螢幕截圖 → OCR** — 用戶閱讀時，點擊觸發 OCR 辨識當前頁面
3. **保留壓縮檔匯入** — 用戶自有的漫畫檔案仍可直接匯入

### 用戶使用流程
1. 從「日漫來源」列表選擇平台（如少年ジャンプ+）
2. 在 WebView 中搜尋並閱讀日文原版漫畫
3. 看到不懂的文字時，點擊「OCR」按鈕
4. App 截取當前畫面 → VisionKit OCR → 日文分詞 → 翻譯
5. 顯示學習面板（日文原文 + 假名注音 + 中文翻譯）

### 與壓縮檔模式的整合
- 壓縮檔模式：直接讀取圖片 → OCR → 學習面板
- WebView 模式：截圖 → OCR → 學習面板
- 兩者共用同一個 OCR → 分詞 → 翻譯 pipeline
